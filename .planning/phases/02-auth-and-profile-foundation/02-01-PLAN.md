    **1. Diagnose and fix the Codable decode mismatch in NetworkService.swift:**

    The backend likely returns snake_case JSON keys (e.g., `first_name`, `last_name`, `resume_file_name`, `work_authorization`, `zip_code`, `work_history`, `job_preferences`, `stripe_customer_id`, `referral_code`, `other_links`, `transcript_file_name`, `remote_only`, `open_to_relocation`, `salary_min`, `start_date`, `end_date`, `veteran_status`, `disability_status`). The Swift model uses camelCase properties with no `CodingKeys`. The default `JSONDecoder()` on line 144 requires exact key matches, so all fields silently decode to nil.

    **Fix approach A (preferred -- global snake_case strategy):**
    In `NetworkService.swift`, in the `execute<T>` method (line ~143-144), change:
    ```swift
    return try JSONDecoder().decode(T.self, from: data)
    ```
    to:
    ```swift
    let decoder = JSONDecoder()
    decoder.keyDecodingStrategy = .convertFromSnakeCase
    return try decoder.decode(T.self, from: data)
    ```

    ```swift
    req.httpBody = try JSONEncoder().encode(body)
    ```

    ```swift
    let encoder = JSONEncoder()
    ```

    The full method should look like:
    ```swift
    func uploadResume(data: Data, fileName: String) async {
        isSaving = true
        AppLogger.files.debug("uploadResume: \(fileName) (\(data.count) bytes)")
        do {
            try await FileUploadService.shared.uploadResume(data: data, fileName: fileName)
            profile.resumeFileName = fileName
            let userId = try await AuthService.shared.getCurrentUserId()
            let _: MessageResponse = try await network.request(
                "/users/\(userId)/profile",
                method: "POST",
                body: profile
            )
            AppLogger.files.info("uploadResume: profile updated with resume key")
            // Re-fetch to confirm persistence and sync server-side transformations
            await fetchProfile()
        } catch {
            AppLogger.files.error("uploadResume: failed -- \(error.localizedDescription)")
            self.error = error.localizedDescription
        }
        isSaving = false
    }
    ```
